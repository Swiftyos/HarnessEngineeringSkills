# Skills Index

## Purpose

This directory contains one subdirectory per installable skill, with each skill
packaging its instructions, references, and helper scripts together.

## File conventions

- Each child directory should represent a single skill and include a `SKILL.md`
  plus an `INDEX.md`.
- Skill assets should stay grouped with the skill that owns them rather than in
  shared catch-all folders.
- Do not commit editor state or transient build output here.
