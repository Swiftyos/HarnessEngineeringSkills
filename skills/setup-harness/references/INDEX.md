# references Index

## Purpose

This directory stores reusable content templates that the `setup-harness` skill
adapts to a user's project when scaffolding docs, scripts, tests, and workflows.

## File conventions

- Files are Markdown reference docs organized by output type.
- Templates should use placeholders only where project-specific details must be
  substituted during scaffolding.
- Reference files should describe structure and constraints, not project-local
  implementation details for this repo.

## Files

- [doc-templates.md](doc-templates.md) - Templates for docs and directory indexes.
- [script-templates.md](script-templates.md) - Templates for bootstrap and docs-quality scripts.
- [test-templates.md](test-templates.md) - Templates for initial smoke or equivalent tests.
- [workflow-templates.md](workflow-templates.md) - Templates for PR, CI, and maintenance workflows.
