# setup-harness Index

## Purpose

This directory defines the `setup-harness` skill and bundles the validator,
reference templates, and instructions needed to scaffold an agent-first repo.

## File conventions

- `SKILL.md` is the canonical instruction file for the skill.
- Reference material belongs in child directories and should stay in Markdown or
  executable script formats appropriate to their purpose.
- Changes here should keep the skill instructions, templates, and validators in
  sync.

## Files

- [SKILL.md](SKILL.md) - Main skill instructions and workflow.

## Subdirectories

- [references/INDEX.md](references/INDEX.md) - Content templates for generated project files.
- [scripts/INDEX.md](scripts/INDEX.md) - Bundled validation scripts copied into target repos.
