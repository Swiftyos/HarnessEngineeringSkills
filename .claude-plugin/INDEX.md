# .claude-plugin Index

## Purpose

This directory holds the Claude plugin metadata that describes this repository
as an installable skills bundle.

## File conventions

- Files here are JSON metadata consumed by Claude-compatible plugin tooling.
- Metadata should stay machine-readable and reflect the current published skill
  package shape.
- Do not place general docs or implementation assets in this directory.

## Files

- [marketplace.json](marketplace.json) - Marketplace metadata for distribution and discovery.
- [plugin.json](plugin.json) - Core plugin manifest for the skills bundle.

## Subdirectories

- No tracked subdirectories.
