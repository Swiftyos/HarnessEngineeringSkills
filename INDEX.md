# Repository Index

## Purpose

This repository stores reusable Harness Engineering skills and the reference
material that teaches agents how to scaffold agent-friendly repositories.

## File conventions

- Top-level files are repo metadata or contributor entrypoints in Markdown.
- Directory-specific implementation details live under each directory's own
  `INDEX.md` and related files.
- Generated artifacts, editor state, and machine-local files should not be
  committed at the repo root.

## Files

- [LICENSE](LICENSE) - Repository license.
- [README.md](README.md) - Human-oriented overview of the skills in this repo.

## Subdirectories

- [.claude-plugin/INDEX.md](.claude-plugin/INDEX.md) - Claude plugin metadata for publishing this skills bundle.
- [skills/INDEX.md](skills/INDEX.md) - Installed skills and their bundled assets.
